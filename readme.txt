A template for building Höyry blocks with modern patterns.

== Description ==

This is a starter template for building Höyry WordPress Gutenberg blocks. It demonstrates the following patterns:

- Parent/child block structure with InnerBlocks
- Server-side rendering via render_callback
- Blockstyle block for visual customization
- Class-based PHP with namespaces
- Custom block category registration
- Internationalization with text domain and script translations
- Modern apiVersion 3 blocks

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/hoyry-block-template` directory
1. Run `npm install` and `npm run build`
1. Activate the plugin through the 'Plugins' screen in WordPress
